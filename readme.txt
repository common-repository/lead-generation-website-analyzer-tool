== Description ==
A must have for any website design, marketing agency, SEO or creative company - This free lead generation plugin adds a website analyzer tool/widget via shortcode anywhere on your website and displays the results of submitted sites.

Requires Google Page Speed Insights API Key which is free to obtain.

== Installation ==

1. Upload the entire `jmj-seo` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Find 'Website Grader' in your admin menu and fill in requested options.
4. Valid Google Pagespeeds API Key is required.  This is free - https://developers.google.com/speed/docs/insights/v4/first-app.
5. Add [jmj-seo] shortcode to the page you want the website analyzer to display on.
